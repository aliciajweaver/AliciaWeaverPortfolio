Girls Who Code (July 2017 - Present)

"I joined Girls Who Code this summer because I wanted to explore more into the world of STEM and Computer Science, learn the different languages, and apply it to hardware to see it come alive."

Week 1

We learned the basics coding priciples such as strings, booleans, conditionals, variables, functions, etc. and applied them to make games in Scratch.

Week 2

We explored coding with Python to make program that produces a picture with a specific color scheme depending on the RBG patterns of the pixels in the original photo.

Week 3

We used django to create an user responsive blog and visited Huge and Ford Innovation!
